## Preview

## Installation

1. copy `zeditor.dcol` to `~/.config/hyde/wallbash/Wall-Ways/`

```bash
cp zeditor.dcol ~/.config/hyde/wallbash/Wall-Ways/
```

2. run `Hyde reload` in the terminal

```bash
Hyde reload
```

3. Enjoy
